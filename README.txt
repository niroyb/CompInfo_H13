Compétition informatique Informatique, Logiciel Hiver 2013
==========================================================

Chaque problème est dans son propre dossier.
Les tests cases associés sont nommés in%d.txt et out%d.txt avec la même valeur pour %d

splitter.py : Divise le fichier contenant tous les cas de tests en deux fichiers par cas
testing frame.py : Test les programmes avec les cas de tests dans leur dossier

Résumé:
--------
* Challenge Appel gagnant: Regex, substring, puzzle, roman numerals
* ChooChooTrain : Permutations, eval
* Combien de Triangles : Adjacency matrix, prevent duplicates
* Les Allumettes : Brute force, search space cutting
* MasterMind helper : Search space reduction
* Profondeur Collatz : Big ints, speed
